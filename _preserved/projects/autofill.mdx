## Overview

Gmail Code Autofill is a Chrome extension that reads numerical verification codes from your Gmail inbox and fills them into web forms automatically. It uses Gmail's API in read-only mode and handles everything locallyâ€”nothing is stored, logged, or sent externally. The extension only requests access to your Gmail data for the explicit purpose of extracting verification codes to improve your login experience.

## Data Usage

This extension:
- Only accesses Gmail messages to extract verification codes (OTP, 2FA)
- Processes all data locally in your browser
- Never transmits your data to any external servers
- Does not track your browsing activity or personal information
- Requests minimal permissions necessary for functionality

## Key Features

- **Autofill Verification Codes** from Gmail into matching form fields
- **Manual Override UI** via popup
- **No Data Storage or Transmission** for maximum privacy
- **Built-in Test Page** for fast debugging

## Tech Stack

JavaScript (ESModules), Gmail API, Chrome Extension APIs (Manifest V3)

## Security

- OAuth-based Gmail access (readonly)
- No external servers, tracking, or analytics
- Local-only parsing and injection
